  ## About / عن التطبيق

  **PreDM** is a professional mobile application for monitoring pre-diabetes risk factors, built according to **ADA 2026 guidelines**.

  تطبيق احترافي لرصد عوامل خطر مرحلة ما قبل السكري، مبني وفق معايير **الجمعية الأمريكية للسكري ADA 2026**.

  ---

  ## Features / المميزات

  - **Risk Assessment** - تقييم مخاطر السكري وفق ADA 2026
  - **Blood Glucose Tracking** - تتبع مستوى سكر الدم
  - **BMI & Weight Monitoring** - مراقبة مؤشر كتلة الجسم والوزن
  - **Exercise Logging** - تسجيل النشاط البدني
  - **Blood Pressure Tracking** - تتبع ضغط الدم  
  - **Medication Alerts** - تنبيهات الأدوية المؤثرة على السكر (كورتيكوستيرويدات، مثبطات المناعة)
  - **Health Education** - تثقيف صحي متكامل
  - **Carbohydrate Calculator** - حاسبة الكربوهيدرات
  - **Electronic Health Record Link** - ربط بالسجل الصحي الإلكتروني

  ---

  ## ADA 2026 Criteria / معايير التشخيص

  | Test | Prediabetes Range |
  |------|------------------|
  | Fasting Glucose | 100-125 mg/dL |
  | 2-Hour Glucose | 140-199 mg/dL |
  | HbA1c | 5.7-6.4% |

  ## Tech Stack

  - **Frontend**: React Native + Expo SDK 54
  - **Navigation**: Expo Router (file-based)
  - **State**: React Context + AsyncStorage
  - **UI**: Custom design system with Nunito font
  - **Icons**: Expo Vector Icons
